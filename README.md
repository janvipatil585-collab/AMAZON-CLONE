# AMAZON-CLONE
A front-end project focused on recreating the Amazon.com homepage interface. This project demonstrates core web development skills by building a complex, multi-section UI from scratch without the use of external frameworks or libraries.

🛠️ Key Features
Hero Section: Responsive banner featuring marketing typography and product imagery.

Grid Layout: A clean, four-column category grid displaying product cards for Clothes, Electronics, Furniture, and more.

Navigation Bar: Replicated top-tier navigation including the search bar, localized delivery indicator, and user account links.

Pixel-Perfect Styling: Extensive use of CSS for custom spacing, typography, and hover effects to mimic the authentic Amazon user experience.

💻 Technologies Used
HTML5: Semantic structure for high accessibility and SEO-friendly layout.
CSS3: Custom properties, Flexbox, and CSS Grid for layout management and responsive design elements.
